Proof Upload & Admin Review / رفع الإثبات ومراجعة الإدارة
طلب روان

ثم يرفع إثبات أنه فتح الحساب — مثل لقطة شاشة، أو رقم الحساب، أو البريد الذي وصله من الوسيط. بعد إتمام الخطوات تنحسب المهمة كمكتملة.

كل خطوة يكون إلها حالة: لم يبدأ، قيد التنفيذ، مكتمل، بانتظار مراجعة، يحتاج إعادة رفع إثبات

ردنا

Plan: Two-tier verification system:

  1. OpenAI Vision API auto-check: When the student uploads a screenshot, our backend sends it to GPT-4o Vision with a prompt asking: "Is this a legitimate Equiti broker confirmation email? Check for: (a) sender from equiti.com/salesforce.com, (b) IB account linking confirmation text, (c) account number reference." The AI returns a confidence score.
  2. Confidence ≥ 90% → Auto-approved instantly. Student proceeds to next step.
  3. Confidence < 90% → Queued for admin/support review. Admin sees the AI's analysis alongside the image.

This reduces admin workload — most legitimate proofs auto-approve, only edge cases need human review.
